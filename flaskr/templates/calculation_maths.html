Character Generation Process
Calculate basic information (age, name, clan, max skill level according to generation, etc..)
Set all atributes to 1
Set all skills to 1
Set all disciplines to 0
Spend XP until it runs out
Cleanup character sheet
XP Calculation
the current formula is:
xp = max(300 (a startup minimum value), generation based max xp times age/max age normalized between 0,2 to 1)
